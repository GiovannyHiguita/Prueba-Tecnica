Prueba Técnica Desarrollador Junior
Gestión de Tareas

Descripción
Este proyecto es una aplicación web de gestión de tareas y perfil de usuario, desarrollada con tecnologías modernas y herramientas de desarrollo innovadoras.
Instalación Local

Pasos para instalar y ejecutar el proyecto
-Clona el repositorio utilizando Git: git clone https://github.com/GiovannyHiguita/Prueba-Tecnica/upload
-Instala las dependencias necesarias: npm install
-Configura las variables de entorno según el archivo .env.example
-Crea la base de datos MongoDB según las instrucciones en .env.example
-Ejecuta el proyecto: npm start
